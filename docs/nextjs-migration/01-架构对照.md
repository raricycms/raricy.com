# 架构对照：Flask ↔ Next.js

一张表看懂概念怎么平移。左边是你现在的 Flask 世界，右边是 `web-next/` 的对应物。

| Flask / 现状 | Next.js / web-next | 说明 |
|---|---|---|
| `run.py` + `create_app()` 应用工厂 | Next 内置，无需工厂 | 配置走 `.env` + `next.config.mjs` |
| 蓝图 `xxx_bp` + `url_prefix` | `src/app/<路径>/` 目录路由 | App Router 用文件系统即路由 |
| Jinja2 模板 `templates/` | Server Components（`page.tsx`）| 服务端渲染 HTML，语义几乎 1:1 |
| `base.html` 母版 + block | `app/layout.tsx` + `children` | 防闪烁主题脚本已平移到 layout |
| 视图函数 `views.py`（返回 HTML）| `page.tsx`（async Server Component）| 直接 `await` 服务层取数 |
| 前端 API `api_views.py`（返回 JSON）| `app/api/**/route.ts` | `{ code, message, ... }` 格式保留 |
| SQLAlchemy 模型 `app/models/*` | `prisma/schema.prisma` | 20 表 1:1 映射，见 [03](03-数据库映射与陷阱.md) |
| `db.session` / `Model.query` | `prisma.<model>.*` | `src/lib/db.ts` 单例 |
| 服务层 `app/service/*`、`blog/services/*` | `src/lib/*-service.ts` | 纯函数 + 显式参数，风格照搬 |
| Flask-Login + `session_version` | `src/lib/session.ts` + `auth.ts` | JWT 会话，失效机制对齐，见 [04](04-认证与会话迁移.md) |
| `@authenticated_required` / `@admin_required` | `getCurrentUser()` + 角色 helper | 在 route/page 里显式判权 |
| werkzeug 密码哈希 | `src/lib/password.ts` | scrypt/pbkdf2 双向互通（已实测）|
| 多处内存限频 | `src/lib/rate-limit.ts` | 抽成共享工具 |
| 自定义 Jinja 过滤器 `datetime_format` | `src/lib/format.ts` | `ymd()` / `categoryFullPath()` 等 |
| `static_url()` 缓存破坏 | Next 资源指纹内置 | 无需手写 |
| SCSS → `main.min.css` | 本切片用 `globals.css`（令牌照搬）| 完整组件化样式为后续工作 |
| 头像 `/auth/avatar/<id>`、图床 `/image/*` | 过渡期 `next.config` rewrites 回源 Flask | 见 `next.config.mjs` |
| 账户微服务 `AccountClient`（httpx）| 待写 TS 版（fetch）| 后端 FastAPI 不变 |
| Turnstile 人机验证 | 待接 `@marsidev/react-turnstile` 或原生 | 注册页用 |
| Markdown：博客客户端渲染 marked+DOMPurify | 待平移为客户端组件 | 逻辑可 1:1 复用 |
| Markdown：故事服务端渲染（Python markdown）| 待用 `remark`/`markdown-it` | 服务端渲染 |

## 目录结构（现状）

```
web-next/
├── prisma/
│   ├── schema.prisma          # 20 表规范映射
│   └── dev.db                 # 规整后的开发副本（gitignored）
├── scripts/
│   └── normalize-datetimes.mjs
├── src/
│   ├── lib/
│   │   ├── db.ts              # Prisma 单例
│   │   ├── password.ts        # werkzeug 兼容哈希（scrypt/pbkdf2）
│   │   ├── session.ts         # JWT 会话
│   │   ├── auth.ts            # getCurrentUser + 角色/禁言判定
│   │   ├── rate-limit.ts      # 共享内存限频
│   │   ├── format.ts          # 响应格式 & 展示辅助
│   │   └── blog-service.ts    # 博客业务逻辑
│   └── app/
│       ├── layout.tsx / globals.css
│       ├── page.tsx           # 首页
│       ├── login/page.tsx     # 登录
│       ├── blog/page.tsx      # 列表
│       ├── blog/[id]/page.tsx # 详情
│       ├── components/        # Navbar, ThemeToggle
│       └── api/
│           ├── auth/{login,logout,me}/route.ts
│           └── blogs/{route, [id]/route, [id]/like/route}.ts
├── next.config.mjs / postcss.config.mjs / tsconfig.json
└── package.json
```
