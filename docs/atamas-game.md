# ATÅMAS — 圆盘组合游戏

## 简介

ATÅMAS（ATOMAS）是一个策略型圆盘组合游戏。玩家在圆形环上放置数字和加号元素，通过巧妙布局触发链式合并产生更大数字，挑战最高分。

灵感来源：ATOMAS 原版游戏。

## 核心玩法

### 基本规则

1. **圆环布局**：元素均匀分布在圆形环上，环上最多容纳 21 个元素。
2. **数字生成**：每回合从中心生成一个数字（初始范围 1-2，随回合数增长）。
3. **加号生成**：每隔 2-8 回合随机生成一个加号（`+`），有 10% 概率为"特殊加号"。
4. **放置操作**：点击圆环上元素之间的空隙即可放置当前中心元素。
5. **合并机制**：
   - **普通加号**：合并左右两侧**相同数字**，结果 = max(a,b) + 1
   - **特殊加号**（黑金 `+`）：合并左右两侧**任意两个数字**，结果 = max(a,b) + 1
6. **链式合并**：合并产生新数字后，如果该数字两侧形成回文模式（A-X-A），自动继续合并 → X+1
7. **游戏结束**：环满 21 个元素且无任何可触发合并的加号。

### 计分

- 每次合并获得被合并元素的值之和作为分数
- 同时追踪"最大版值"（Max Plate）

### 撤销

- 每 25 回合可撤销一次上一步操作
- 撤销按钮显示冷却剩余回合数
- 动画播放中无法撤销

## 界面功能

| 区域 | 功能 |
|------|------|
| 画布中心 | 显示待放置元素（数字或加号），带呼吸动画 |
| 右侧预览面板 | 显示接下来 3 个元素（可折叠） |
| 顶部栏 | 标题、语言选择器（30+ 语言）、明暗模式切换 |
| 信息面板 | Reset（重置）、Recall（撤销）、当前操作提示 |
| 底部状态栏 | 最大值、分数、警告指示灯（18+ 元素时亮起） |
| 悬停指示 | 鼠标悬停圆环时显示绿色目标位置指示器 |

## 技术架构

### 文件结构

```
app/templates/game/atamas.html    # Jinja2 模板（继承 base.html）
app/static/js/game/atamas.js      # 游戏核心逻辑（~2200行）
app/static/scss/pages/_atamas.scss # 游戏样式
```

### 渲染方式

使用 Canvas 2D API 进行全部游戏渲染，不使用 DOM 元素布局。动画使用 `requestAnimationFrame` 驱动。

### 状态管理

游戏状态集中存储在 `state` 对象中：

- `elements[]` — 环上所有元素
- `pendingElement` — 中心待放置元素
- `pendingQueue[]` — 预览队列（预生成后续元素）
- `round` — 当前回合数
- `gameOver` / `animating` — 状态标志

### 语言系统

内置 30+ 种语言的翻译表（`TRANSLATIONS` 对象），通过 `localStorage` 持久化语言偏好。UI 文本通过 `getTranslation(key)` / `formatTranslation(key, params)` 动态获取。

### 明暗模式

独立的明暗模式切换，通过 `.game-atamas-page` 容器的 `.light-mode` class 控制。CSS 变量未与项目主题系统对接（待后续统一）。

## 路由

- URL：`/game/atamas`
- 蓝图：`game_bp`
- 不需要登录即可游玩

## 注意事项

1. 游戏使用 `Math.random()` 生成随机数，非加密安全随机。
2. 所有游戏状态仅存在于浏览器内存中，刷新页面即丢失。
3. 原文件中的 localStorage 登录/排行系统已移除，后续可对接项目用户系统。
4. 预览队列预生成 5 个元素（1 个在中心 + 3 个预览 + 1 个缓冲），确保预览准确。
