# 部署与运行

## 1. 环境要求

- Python 3.8+
- Node.js LTS（用于 SCSS 编译，可选但推荐）
- SQLite（默认，无需额外安装）

## 2. 快速初始化

```bash
# 安装 Python 依赖
pip install -r requirements.txt

# 安装 Node 依赖（SCSS 编译用）
npm install

# 配置环境变量
cp example.env .env
# 编辑 .env，填入 SECRET_KEY 等必要配置

# 创建目录并初始化数据库
python check_instance.py
flask db upgrade
```

## 3. 配置文件（.env）

```ini
SQLALCHEMY_DATABASE_URI=sqlite:///../instance/database/db.db
SECRET_KEY=<替换为强随机字符串>
DEBUG=True
TURNSTILE_SITE_KEY=your_site_key
TURNSTILE_SECRET_KEY=your_secret_key
TURNSTILE_AVAILABLE=False
CONFIG_TYPE=development
```

通过 `CONFIG_TYPE` 切换环境：`development` / `production` / `testing`。

## 4. 启动

```bash
python run.py
```

## 5. 管理员设置

```bash
flask promote-admin <username>     # 授予管理员权限
flask promote-owner <username>     # 授予站长权限
flask demote-admin <username>      # 撤销管理员权限
flask demote-owner <username>      # 撤销站长权限
```

用户管理页面：`/auth/user_management`

## 6. 数据库迁移

日常使用 `flask db upgrade` 即可。如需生成新迁移文件：

```bash
flask db migrate -m "描述"
flask db upgrade
```

## 7. 故事（Stories）配置

故事文件放在 `instance/stories/` 下，每个子目录为一个"小说集"。

### 小说集 info.json

```json
{
    "title": "其它小说",
    "name": "转载/代发",
    "details": "聪明山朋友的原创故事。",
    "description": "在这里阅读转载的精彩小说",
    "ignore": false,
    "priority": 30
}
```

- `priority` — 越大越靠前
- `ignore: true` — 不在列表中显示

### 故事 Markdown 文件头

```markdown
---
title: 聪明山庄
author: raricy
genre: 短篇小说
description: 小r想睡觉。帮帮它。
ignore: false
priority: 96
---

# 正文开始
```

## 8. 样式构建（SCSS → CSS）

开发模式（监听文件变化）：

```bash
sass --style=expanded --source-map static/scss/main.scss static/css/main.css --load-path=static/scss --watch
```

生产模式（编译 + 前缀 + 压缩）：

```bash
sass --style=expanded static/scss/main.scss static/css/main.css --no-source-map --load-path=static/scss
postcss static/css/main.css --use autoprefixer --no-map | postcss --use cssnano --no-map > static/css/main.min.css
```

详见 [前端样式构建指南](前端样式构建指南.md)。

## 9. 生产部署要点

- 设置 `CONFIG_TYPE=production`
- 设置 `DEBUG=False`
- `SECRET_KEY` 使用强随机字符串
- 配置 Nginx 反向代理（应用已启用 `ProxyFix` 信任 `X-Forwarded-*` 头）
- 启用 Turnstile 验证（`TURNSTILE_AVAILABLE=True` 并填入有效密钥）
- 生产环境模板引用 `static/css/main.min.css`
